# Tomasino Blog
A simple blog website.

## Website
http://tomasinoblog.atwebpages.com/
or
https://tomasinoblog.000webhostapp.com/

## Features
- Create a Post
- View All Posts
- View a Single Post
- Add Comment
- Update a Single Post
- Delete a Single Post
- Uses a MySQL Database

## Language/s and software used
- HTML
- CSS
- Bootstrap
- PHP
- MySQL
- XAMPP
